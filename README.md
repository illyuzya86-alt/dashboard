# Универсальный дашборд для анализа данных из Excel

Интерактивный одностраничный дашборд, который автоматически анализирует любой Excel-файл и строит визуализации.

## Как это работает

1. Загрузите Excel-файл (.xlsx) через кнопку "Загрузить Excel" или перетащите файл в окно браузера
2. Дашборд автоматически:
   - Определяет структуру таблицы (заголовки, строки, колонки с данными)
   - Строит 4 интерактивных графика
   - Рассчитывает KPI
   - Создаёт фильтры по строкам и колонкам
   - Отображает таблицу с сортировкой и поиском

## Требования к Excel-файлу

- Файл должен содержать таблицу на первом листе
- **Первая строка** — заголовки колонок
- **Первая колонка** — названия строк (текст)
- **Остальные колонки** — числовые данные
- Колонка "Итого"/"Всего" (если есть) будет автоматически исключена из данных

**Пример структуры:**

| Программа | Когалым | Лангепас | Сургут | Итого |
|-----------|---------|----------|--------|-------|
| Курс А | 10 | 15 | 20 | 45 |
| Курс Б | 5 | 8 | 12 | 25 |

## Возможности

- **Универсальный парсер** — работает с любыми Excel-таблицами, автоматически определяет структуру
- **KPI-панель** — общая сумма, топ-3 строк, топ-3 колонок, среднее/медиана, число строк > порога
- **4 графика Plotly.js**:
  - Столбчатая диаграмма сумм по строкам
  - Круговая диаграмма распределения по колонкам
  - Детализация по выбранной колонке
  - Тепловая матрица (строки vs колонки)
- **Таблица** с сортировкой по колонкам, поиском, экспортом CSV/Excel
- **Фильтры** по строкам и колонкам (multi-select), по длительности (часы)
- **Drag & drop** — перетащите файл в окно браузера
- **Экспорт дашборда в PDF**
- **Адаптивная верстка** (mobile-first)

## Установка и запуск

```bash
# Открыть index.html в браузере
start index.html
```

Для корректной загрузки sample.json используйте локальный сервер:
```bash
python -m http.server 8000
# http://localhost:8000
```
